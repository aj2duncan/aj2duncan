Moving to Github Pages and Jekyll

April 1, 2015   

I started this website out on wordpress.com and then moved it to a server where I was paying too much money and not getting what I wanted.

Recently I’ve started using git as part of my research work and therefore heard about GitHub and finally about GitHub Pages.

So long story short I have moved my website to Github Pages and used Jekyll to make blogging easier. Hopefully this will mean more posts and more pages for my students.

I won’t bore you with details at present but will hopefully put something together documenting how I did it, as much for my benefit as for anyone else’s.
